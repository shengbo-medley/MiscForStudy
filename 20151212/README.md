- data_plot.R

- hw_data.csv
 - 身長・体重データ
- matching_data.csv
 - 身長・体重データ・マッチングデータ
- matching_predict.R
 - 身長・体重データ・マッチングデータから、サポートベクターマシンを使って、身長・体重データからマッチングするかを推測するサンプル
- matching_predict_data.csv
 - マッチングするかどうか予測するための身長・体重データ
- predict_data.cs
 - 身長を予測するための体重データ
